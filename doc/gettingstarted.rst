
Getting Started
===============

Install
-------

:ref:`wheezy.security` requires `python`_ version 3.6+. It is independent of
operating system. You can install it from `pypi`_ site ::

    $ pip install wheezy.security

.. _`pypi`: http://pypi.python.org/pypi/wheezy.security
.. _`python`: http://www.python.org
